# Ghi chu thiet ke frontend

## Nguon tham khao

- RefRef: tham khao mo hinh referral program, participant/referrer, reward rule, reward approval, analytics va activity log.
- Shadcn dashboard templates: tham khao cach to chuc sidebar, card thong ke, data table, chart va trang settings.
- Affiliate tracking source: tham khao luong tracking link, conversion/postback va bang thong ke click/conversion.

## Anh xa sang bai toan CTV XKLD

- Referrer/affiliate -> CTV.
- Referral/conversion -> Khach hang duoc gioi thieu va dat cac moc ho so.
- Reward rule -> Chuong trinh diem.
- Reward approval -> De nghi cong diem can Admin/CTV xac nhan.
- Analytics -> Dashboard hieu suat, diem thang, KPI va luong.
- Fraud monitoring -> Canh bao trung thong tin, diem tang bat thuong, ho so thieu minh chung.

## Pham vi hien tai

Frontend hien tai la prototype React JS chay doc lap voi data mau. Cac thao tac nhu duyet/tui choi diem duoc xu ly tren state local de minh hoa luong UI. Khi co backend, cac man hinh nay co the noi API theo cac nhom endpoint: CTV, customer, point program, point request, activity va payroll.
