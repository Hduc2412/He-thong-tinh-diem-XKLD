# He thong tinh diem CTV XKLD

Frontend duoc dat trong thu muc `frontend`.

## Cong nghe

- React JS voi Vite
- Tailwind CSS
- UI component theo phong cach shadcn/ui
- Lucide React cho icons
- Recharts cho bieu do dashboard

## Chay frontend

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
cd frontend
npm run build
```
